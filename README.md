# 映えなイイネ

## サイト概要
### サイトテーマ
「映えない」写真を投稿するSNSサイト
- ゲストログイン
- 会員機能
- 投稿機能
- 投稿検索機能(キーワード検索)
- ユーザー管理機能(管理者)
- 映えなイイネ(ブックマーク)機能
- 映えなイイネ(ブックマーク)一覧表示機能
- フォロー機能
- コメント機能
- ユーザ検索機能

### テーマを選んだ理由
「映える」写真だけでなく、「映えない」写真も投稿して讃え合う場があってもいいのではないか..？と考えたからです。
